# Student Management System (Python - Console Application)

This project is a simple Student Management System developed in Python for a console enviroment.
The program allows the user to manage student information using basic data structure such as lists and dictionaries.

The system includes the main CRUD operations:
* Add students
* View the list of students
* Search students by ID
* Update student information
* Delete students

The system also includes:
* Error handling
* A functional menu with options dislayed
* Double confirmation to delete a student
* Check duplicated and inexistents IDs

How to execute it?
Just clone the repository, execute "python3 main.py" and suit yourself!
